Qvico

Sistema de Gestión para la vigilancia y Control de las Juntas de Acción Comunal.

Qstats

Esta herramienta permite el manejo, control y consulta de indicadores, visualizándolos a través de Matrices, gráficos estadísticos y mapas. Administrable y flexible.

Qmap

Sistema de Información que permite mostrar la información de bases de datos, hojas de cálculo, documentos, etc., en mapas de fácil manejo, brindando a los usuarios una poderosa herramienta para la toma de decisiones.
